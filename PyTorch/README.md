# Công nghệ sử dụng

## Bài lab được xây dựng và thực hiện với các công nghệ sau:

- Python: Ngôn ngữ lập trình chính dùng để xử lý và phân tích dữ liệu.

- NumPy: Hỗ trợ tính toán số học và làm việc với mảng dữ liệu.

- Pandas: Dùng để đọc, xử lý và phân tích dữ liệu dạng bảng (DataFrame).

- Matplotlib / Seaborn (nếu có): Trực quan hóa dữ liệu thông qua các biểu đồ.

- Jupyter Notebook: Môi trường làm việc giúp chạy code, quan sát kết quả và giải thích từng bước.

## Đọc và hiển thị ảnh

Đọc ảnh từ file bằng Pillow và OpenCV.

Hiển thị ảnh ra màn hình.

Nhận xét sự khác nhau về thứ tự kênh màu (RGB và BGR).

Hiển thị ảnh sau khi cắt.
